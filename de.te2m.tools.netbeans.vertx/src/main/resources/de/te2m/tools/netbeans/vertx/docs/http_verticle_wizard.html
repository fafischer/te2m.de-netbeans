Web Server Verticle Wizard

This wizard creates a new Web Server verticle.

Select Wizard

Select the wizard.

Define Verticle

Define the Vertcle. Enter all required information about it.

By enabling the related check box also a (empty) test case will be created for it.

Check Dependencies

This verticle has an additional dependency to vertx-web. The POM-File will not be modified automatically. Thats why it has to be checked after generating the verticle.

Sample: 
        <dependency>
            <groupId>io.vertx</groupId>
            <artifactId>vertx-web</artifactId>
            <version>${vertx.version}</version>
        </dependency>
